Bu proje, spor salonu yönetim sistemidir. Sistem, birden fazla spor salonu lokasyonunu yönetebilen, üyelik talepleri, randevu planlaması, antrenör atamaları ve yapay zeka destekli antrenman planları oluşturma gibi özellikleri içermektedir. Uygulama, Clean Architecture prensiplerine uygun olarak tasarlanmış olup API ve MVC katmanlarından oluşmaktadır. Kullanıcı rolleri Admin, GymOwner ve Member olarak ayrılmıştır. Admin tüm sistem üzerinde tam yetkiye sahipken, GymOwner sadece kendi salonuna ait üyeleri, antrenörleri, hizmetleri ve randevuları yönetebilmektedir. Member rolündeki kullanıcılar ise üyelik talebinde bulunabilir, randevu alabilir ve AI destekli kişiselleştirilmiş antrenman planları oluşturabilmektedir. Sistem PostgreSQL veritabanı kullanmakta ve Entity Framework Core ile veri erişimi sağlamaktadır.
